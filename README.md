# LIMB

Code for a robot limb manipulation toolkit in C++. Developed in Arduino IDE.



Toolbox takes the robot limb's Denavit-Hartenberg table and uses it to calculate Kinematics using Homogeneous Transformation Matrices.



Currently, no Dynamics tools are implemented.



Dependencies:

-Matrix.h (https://github.com/the-real-Nathanael-G/MATRIX)
